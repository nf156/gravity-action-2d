gravity-action-2d

## 使用技術 / 開発環境
===

# 

# \- Language: C++

# \- Platform: Windows

# \- Graphics API: DirectX 11

# \- SDK: Windows SDK（Windows 10/11）

# \- IDE: Visual Studio 2022

# 

# \## 使用ライブラリ（現時点）

# 

# 本プロジェクトは、以下の \*\*公式SDKライブラリ\*\* のみを使用しています。

# 

# \- `d3d11.lib`（Direct3D 11）

# \- `dxgi.lib`（DXGI）

# \- `d3dcompiler.lib`（HLSLコンパイル）

# 

# ※ いずれも Windows SDK に含まれる標準ライブラリです。  

# ※ サードパーティ製ランタイムの同梱はありません（現時点）。
